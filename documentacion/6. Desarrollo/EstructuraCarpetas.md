# Estructura de Carpetas

```
codigo/
└── src
    ├── domain
    │   ├── entities
    │   ├── repositories
    │   └── use-cases
    ├── infrastructure
    │   └── database
    ├── application
    │   └── services
    └── interfaces
        └── controllers
```

Esta estructura refleja las capas de Clean Architecture y permite aislar la lógica de negocio de las dependencias externas.
